PATRIMONIUM editor

The PATRIMONIUM editor is an eXist-db application that writes and reads XML/TEI files. It features a front-end interface that can be customized in order to display specific elements of a TEI document. Editing an element is done through user-friendly imputes fields, dropdown lists or other widgets. The app is primarily conceived for the edition and annotation of epigraphic texts and comes with a series of tools to easily encode epigraphic annotations and for metadata enrichment.
